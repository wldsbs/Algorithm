# Algorithm
코딩테스트 문제 풀이

## 커밋 규칙   

### Format   
#### {Keyword}: {출처} {난이도} {제목}

### Keyword
- `solve`: 풀이에 성공한 새 문제 추가
- `fail`: 풀이에 실패한 새 문제 추가
- `improve`: 새로운 풀이 추가
- `docs`: readme 등 문서 수정
